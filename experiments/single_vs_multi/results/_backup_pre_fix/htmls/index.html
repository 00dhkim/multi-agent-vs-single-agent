🎉 멀티에이전트만 성공한 6개 시나리오

실험 한 줄 요약. Toolathlon 벤치마크에서 강력한 단일 에이전트가 실패한 9개 작업 중 6개를 멀티에이전트가 성공시켰다(66.7%). 두 구조 모두 같은 모델(gpt-5)과 같은 도구를 사용했지만, 역할 분리 + 완료 직전 독립 검증/복구 패스가 결정적 차이를 만들었다.

아래 카드를 눌러 시나리오별로, 사람이 이해할 수 있는 수준의 설명을 확인하라.

shoppingInventory Sync — 6개 도시 창고 → WooCommerce 재고 동기화
단일: 0 / 51 통과. 멀티: 51 / 51 통과(100%).
 officePaper Checker — LaTeX 논문의 citation/label 정합성 검사
단일: 일부만 수정. 멀티: 전체 파일을 점검·재참조 정규화.
 officeExcel Data Transformation — 2차원 표를 1차원 레코드로 변환
단일: Processed.xlsx 미생성. 멀티: 양식 대조 후 산출 완료.
 officeArrange Workspace — 18개 폴더 / 22개 파일 구조 재배치
단일: 경로 구조 일치 실패. 멀티: 검증/복구 패스로 정규화.
 officeReimbursement Form Filler — 택시 영수증 PDF → 부서 정산 엑셀
단일: department_expenses.xlsx 미생성. 멀티: 추출·기입·이름 변경 완료.
 eduPPT Analysis — 컴파일러 수업 PPT/HW 요약 NOTE.md 작성
단일: NOTE.md 미생성. 멀티: 핵심 정의·코드·HW 설명 모두 포함.
